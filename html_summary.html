Semantic Tags

  • < nav > : Used for website navigation
  • < main > : Used for the main content of a page
  • < section > : Defines a section in a document. A section is a thematic grouping of content. Examples:
    • Introduction
    • Content
    • Contact Information
  • < article > : Specifies self contained content
  • < footer > : Specifies a footer for a document or section
  • < aside > : Defines content aside from the content it is placed in (e.g. sidebar)

Formatting Tags

There are several content tags that are used for formatting text. These atags are not block tags, This means that they are used inline to format text inside a block.
We can make text bold , strong , italic , or emphasized . Other ways of emphasizing text include highlighting it or underlining it. We can also change text to a smaller size, write it as a subscript , superscript or crossed out .
