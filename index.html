Unit 3

In Unit 3, I learned how to set up and make my own website using both HTML and CSS. First, I was introduced to CSS, how it worked, and how it should be formatted. I learned how to start a code in CSS and some specific vocabulary such as selector and property value. I got use this information to build the basics of my website. I also learned how to add deatil to a website using CSS. I learned about how to arrange color typography of a website so that not only can a website be readable but it can also look appealing. I learned about the box model and how margins, borders, and padding worked together to create the "space" of a website. Finally, I learned about how to create the layout of the website, which was the most frustrating part of the assignment.

Some things I found important were:

  • How to format a website through coding
  • The different types of typography that are the most useful
  • How to add small details to a website

Unit 4

In Unit 4, I was introduced into the use of web graphics. I learned how to use them correctly and where to search for them so that I didn't violate any copyright laws. I also learned how to edit web graphics, like cropping and resizing images, to make it look the best that it could possibly look on my website. In addition, I learned how to create both a button and a banner with the help of Photoshop. I learned how to physically edit images in photoshop to match my website and how to put those buttons and banners onto my website.

Some things I found important were:

  • Finding the rigth web graphics from the right websites
  • How to create a banner
  • The different types of back grounds there are

Unit 5

In Unit 5, I learned more about how to code in Javascript. At the beginning of the lesson, I learned some concepts of Javascript like what client-side scripting was versus what server-side scripting was. After the basics, I learned how to code a multitude of things in Javascript such as how to show an alert, how to debug my page, how to build a Javascript clock, and how to add my own video buttons. In addition, I was able to practice my coding skills by adding my own styles, function, and css code.

Some things I found important were:

  • Learning basic concepts like client-side scripting vs. server-side scripting
  • How to create a clock
  • How to debug my own page
